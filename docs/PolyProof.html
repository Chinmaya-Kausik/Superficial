Computer assisted proof : Homogeneous length functions on groups

A script to replicate finding of the computer assisted proof in the PolyMath 14 project is available as an executable jar here. To run this, you need to have Java 8 installed. Assuming Java 8 is setup, the script can be run from a directory to which it is downloaded by running the following.

java -jar polymath-proof

This outputs status updates and then the proof. The status updates are output to the standard error, so the proof can be saved by just redirecting the standard output, for example run the following.

java -jar polymath-proof > proof.md

The full source is at https://github.com/siddhartha-gadgil/Superficial. The script is generated from the object ProofScript in the file ProofFinder.scala from which one can trace backwards to explore how the proof in this script was generated. During the project we used the file LinearNormProofs.scala instead, but this is less robust due to using more memory and garbage collection (since suboptimal proofs are constructed and discarded).
